## Переменные и их значения
- В Python нет объявления переменных, официально это называется assignment — «назначение, связывание». Мы связываем имя и значение.
- При именовании переменных принято использовать snake_case.
- Что есть литерал?
```python
# Прямо в коде явно указано значение переменной - этой переменной значение
# задано при помощи литерала. В данном случае 15 - числовой литерал.
quantity_of_jars = 15

# А значение этой переменной вычисляется в ходе арифметической операции -
# значение переменной указано неявно, через выражение.
total_quantity = quantity_of_jars + 10

print(total_quantity)
```
## Типизация и типы данных
- Python относится к языкам с **динамической** и **сильной типизацией**.
- Узнать тип переменной можно с помощью функции `type()`.
- В языке Python — сильная типизация. Python не станет сам преобразовывать один тип данных в другой ради того, чтобы выполнить какие-то операции. При необходимости разработчик должен сам преобразовать один тип в другой.
```python
# Тут речь идет про знакомую нам со школы штуку
name = 'Frol'
age = 15
print(name + ' ' + str(age))
```
## Константы
- Константы приянто писать заглавными буквами.
- Константы в Python являются таковыми лишь по смыслу и на основе договорённости, но не технически.
